Your 2 Pure Mitti AI Agents

One agent plans the content, one agent edits it. Set them up once, then use them every day.
What these are

Two AI assistants, built for Pure Mitti

An "AI agent" is a chatbot you've given a fixed set of instructions — its brain — so it always acts like a Pure Mitti expert instead of a blank AI. You set each one up once (below), then just talk to it. Each agent's brain is a text file in this folder; you paste it in during setup.

Agent 1

Pure Mitti Planner

Creates ready-to-shoot daily video plans — hook, shot list, script, caption, hashtags — for Reels, TikTok, Shorts and Facebook. Give it a theme or a week and it hands back briefs the team can film.

brain file → agents/Agent_1_Content_Planner_PASTE.txt
Things to say to it
"Make next week's 7-day video plan."
"Give me 5 hook options for a soap-cutting ASMR video."
"Plan a gifting week for the holidays."
"Turn this product into a Reel idea: [paste product]."
Agent 2

Pure Mitti Editor

Polishes anything already written — scripts, hooks, captions, hashtags. It sharpens the hook, cuts fluff, fixes the voice, and flags risky health claims before you post. Paste a draft, get back a clean final version.

brain file → agents/Agent_2_Editor_PASTE.txt
Things to say to it
"Tighten this caption: [paste]."
"Rewrite this hook to be scroll-stopping: [paste]."
"Check this script for risky health claims: [paste]."
"Trim these hashtags to the best 8: [paste]."
How to set them up

Pick one route — do it twice, once per agent

All three routes do the same thing: give the AI its brain. Route 3 is free and works today.

1

ChatGPT — Custom GPT

Best if you have ChatGPT Plus · gives you a clickable, shareable agent
  1. In ChatGPT, click your name → My GPTs → Create a GPT → Configure.
  2. Name it Pure Mitti Planner (or Editor).
  3. Open the agent's brain file, copy everything, and paste it into the Instructions box.
  4. Add the "Things to say to it" above as Conversation starters. Save → set to "Only me" or share a link.
2

Claude — Project

Best if you have Claude Pro
  1. In Claude, click Projects → Create Project, name it Pure Mitti Planner.
  2. Open Set custom instructions, paste the agent's brain file, save.
  3. Every chat inside that project now behaves as the agent.
3

Free — works in any chatbot, today

No paid account needed · ChatGPT free, Claude free, Gemini, Copilot
  1. Open a new chat.
  2. Open the agent's brain file, copy all of it, paste it as your first message, and send.
  3. The AI is now the agent for the rest of that chat — just start giving it tasks.
Tip: keep both agents open in two tabs. Use the Planner to create a video brief, then paste its script into the Editor to polish it before you film. That two-step loop is the whole workflow.
