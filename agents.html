Set up either agent as a standalone assistant

All three routes do the same thing: give the AI its "brain" — a fixed set of instructions so it always acts like a Pure Mitti expert. Route 3 is free and works today.

1

ChatGPT — Custom GPT

Best if you have ChatGPT Plus · gives you a clickable, shareable agent
  1. In ChatGPT, click your name → My GPTs → Create a GPT → Configure.
  2. Name it "Pure Mitti Planner" (or Editor), then open the brain file below, copy everything, and paste it into the Instructions box.
  3. Add the starter prompts above as conversation starters. Save → set to "Only me" or share a link.
2

Claude — Project

Best if you have Claude Pro
  1. In Claude, click Projects → Create Project, name it "Pure Mitti Planner".
  2. Open Set custom instructions, paste the agent's brain file, save.
  3. Every chat inside that project now behaves as the agent.
3

Free — works in any chatbot, today

No paid account needed · ChatGPT free, Claude free, Gemini, Copilot
  1. Open a new chat.
  2. Open the brain file below, copy all of it, paste it as your first message, and send.
  3. The AI is now the agent for the rest of that chat.
